# About 解説動画カードのテーマ間形状統一 Design

## Goal

Aboutページの「解説動画」カードで、ライト／ダーク切り替え時にモニターや机上機材の形・位置が変わらないようにする。他の活動カードと同様に、ライト版を基準画像としてダーク版の照明と配色だけを変更する。

## Root Cause

現在の`activity-video-light.webp`と`activity-video-dark.webp`は別々にゼロから生成されている。そのため、同じ構図を指示していても、モニター枠、スタンド、机上棚、キーボード、マウス、マイクの形状と配置に差が生じている。

## Scope

- ライト版`frontend/public/images/about/activity-video-light.webp`は変更しない。
- ダーク版`frontend/public/images/about/activity-video-dark.webp`だけを置き換える。
- `About.tsx`の画像パス、HTMLコピー、ボタン、リンク、代替テキストは変更しない。
- 活動カードのレイアウトとCSSは変更しない。

## Image Edit

- ライト版を編集対象として使用する。
- モニター、モニタースタンド、机上棚、キーボード、マウス、マイクについて、形状、位置、比率、輪郭、遠近感を維持する。
- カメラ位置、トリミング、左上のコピー用余白、机と壁の境界を維持する。
- 変更するのは環境の照明、背景と机の配色、画面の控えめな青い発光だけとする。
- モニター画面の山並みと文字なしの音声波形を維持する。
- 人、文字、数字、ロゴ、UIラベル、ボタン、透かしを追加しない。
- 最終画像は既存と同じ1600×1024のWebPとする。

## Theme Behavior

`resolvedTheme`でライト／ダーク画像を選択する既存実装を維持する。ライトテーマでは既存ライト版、ダークテーマでは新しいダーク版を1枚だけ表示する。既存のダーク用フィルター、マスク、コピー可読性グラデーションも維持する。

## Verification

- ライト版と新しいダーク版を高解像度で比較し、すべての機材の形状と配置が一致していることを目視確認する。
- ダーク版が1600×1024のWebPであることを確認する。
- About活動カードのテーマ別画像パスと既存リンクを確認する回帰テストを実行する。
- PCとスマートフォンのライト／ダークで、左上のコピーと右側の機材が重ならず、横スクロールが発生しないことを確認する。
