# Game Community Warm Static Refresh Design

## Goal

`/game-community` から閲覧中に意図せず要素が動く演出を取り除き、手書きの文字と紙のような質感を広げて、放課後に仲間が集まる温かい雰囲気へ整える。

## Motion

- GSAP の初回出現、スクロール連動、カードの段階表示をすべて撤去する。
- ページルートの自動フェードも撤去し、初期表示から全コンテンツを安定して見せる。
- ホバーは入力への反応として、影と色の小さな変化だけ残す。位置移動や回転の上書きは行わない。
- `prefers-reduced-motion` では残るトランジションも停止する。

## Typography and Tone

- 既存の `Yomogi` / `Klee One` を、注釈だけでなくページ見出し、カード見出し、タグ、CTA にも適用する。
- 本文は可読性を優先して通常書体を維持する。
- 英字の大見出しは手書き書体にしつつ、ウェイトを抑えて柔らかな字間にする。

## Visual Treatment

- 明るい背景を白・グレーから生成りと淡いピーチへ寄せる。
- カードを真っ白なUIパネルではなく、少し色味のある紙として見せる。
- 境界線を暖色にし、影は薄い茶系へ変更する。
- ヒーローの濃紺は維持しつつ、コーラルと琥珀色の光を増やして夜の集まりらしい温度感を出す。
- 既存の付箋、テープ、手描き線、ポラロイド風写真を再利用し、装飾の追加はCSSに限定する。

## Scope

- `GameCommunity.tsx` からアニメーション用 ref と hook 呼び出しを削除する。
- ゲームコミュニティ専用CSSを更新する。
- 不要になった `useGameCommunityAnimations.ts` を削除する。
- 他ページの共通アニメーションや配色は変更しない。

## Verification

- TypeScript ビルドと lint を実行する。
- `/game-community` をデスクトップとモバイル幅で表示し、初期表示、スクロール、ダークモード、ホバー、文字の可読性を確認する。
