# About 活動カードのコピー配置統一 Design

## Goal

Aboutページの4つの活動カードで、見出し・説明・ボタンをすべて左上の安全領域へ統一する。数学カードは画像を作り直し、タブレットとペンを右下寄りへ移して、文字と主役が重ならないようにする。

## Layout

- PCでは既存の2列×2行、スマートフォンでは既存の1列4枚を維持する。
- 全カードのコピーは左上に配置する。PCでは左・上34px、スマートフォンでは左・右24px・上28pxを使う。
- 数学カードの右下専用コピー配置と右下用グラデーションは削除する。
- コピー領域のグラデーションは、全カードで左側だけに敷く。画像全体は覆わない。

## Buttons

- 見出し、説明、ボタンの順番と既存の文言・リンク先は変えない。
- `.activity-actions` は説明文の直後ではなく、一定の余白を置く。全カードで左寄せに統一する。
- 2つのボタンを持つ開発カードは、同じ左寄せの行内で折り返せるままにする。
- 数学カードもほかのカードと同じ左寄せボタンにする。

## Images and theme behavior

- `activity-math-light.webp` と `activity-math-dark.webp` を再生成する。
- 2枚は同じカメラ構図・主役配置とし、タブレットとペンを右下寄りに置く。左上には見出し、説明、ボタン用の空き領域を残す。
- 画像内には文字、ロゴ、UIラベル、ボタンを描かない。
- ライト／ダークの画像切替、画像1枚だけの描画、代替テキスト1回の提供は維持する。

## Verification

- 数学カードが左上配置クラスを持つことを自動テストで確認する。
- ライト／ダークの画像パスと、すべての既存リンクを回帰テストで確認する。
- 数学のライト／ダークWebPが1600×1024であることを確認する。
- PCとスマートフォンで、ライト／ダーク双方のカードをブラウザ確認する。ボタンが主役と重ならず、左寄せでそろうことを確認する。
